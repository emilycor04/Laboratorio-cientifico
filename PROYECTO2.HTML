Instrumentos y materiales del laboratorio

El laboratorio es un lugar que cuenta con materiales necesarios para realizar investigaciones, experimentos y trabajos practicos.

¿Por qué el laboratorio es el mejor lugar para realizar nuestro experimentos?

Tiene espacios abundantes para aplicar con toda seguridad nuestro experimentos, tiene una iluminación adecuada para toda clase de actividades. Ademas, podemos descubrir si nuestra teoria es acercatada o no mediante el trabajo practico que en este caso serian los experimentos.

Los instrumentos de laboratorio estan fabricados con materiales como vidrio, metal, madera, porcelana y goma. Cuando se realizan practicas en el laboratorio es necesario conocer los instrumentos del laboratorio, la forma de utilizarlos y lo scuidados que se deben al manipularlos

    Termómetro

    El termómetro es un instrumento que sirve para medir la temperatura ya sea alta o baja, de diversas sustancias y estos pueden ser principalmente líquidos y gaseosos, además puede medir propiedades físicas como lo son el punto de ebullición y punto de congelación.


    Agitador

    El agitador es una varilla regularmente de vidrio, que sirve para mezclar o revolver algunas sustancias o para preparar disoluciones y suspensiones



    Matraz

    Es un recipiente de cristal, con una base plana, un cuerpo en forma de cono y un cuello cilíndrico sirve para contener o calentar las sustancias liquidas, además de mezclar las sustancias agitándolas


    Tubo de ensayo

    Se trata de un pequeño tubo de forma cilíndrica con un extremo cerrado redondeado y el otro abierto. Este tubo es utilizado para resistir las altas temperaturas.




    Mortero

    Es un recipiente circular y viene acompañado de un mazo pesado llamado “Mano o Pilón. Se utiliza para moler, triturar y mezclar sustancias solidas.




    Microscopio

    El microscopio es un instrumento que permite observar objetos que son demasiado pequeños y estos no pueden ser perceptible por el ojo humano.




    Probeta

    Es un tubo alargado y transparente que incluye una graduación o medida. Este instrumento se utiliza para contener y medir un líquido o gas, además con este tubo podemos conocer el volumen de la sustancia que se encuentra en su interior.




    Gradilla

    La gradilla es una herramienta utilizada para sostener de forma vertical varios tubos de ensayo. También para transportar los tubos de ensayo y manipular sus sustancias a la vez evitando posibles derrames.




    Pipeta

    La pipeta es un instrumento de medición y es utilizada para transferir líquidos de un recipiente a otro además de calcular el volumen de las sustancias. Posee una forma cilíndrica y suele permitir un volumen hasta 20 ml.




    Vaso precipitado

    Es un recipiente de forma cilíndrica con un fondo plano. Tiene diversas funciones y algunas de estas son: es utilizada como contenedor, puede calentar un compuesto, transporta sustancias, puede mezclar, diluir y disolver varias sustancias entre otras funciones




    Balanza analitica

    La balanza analítica es un instrumento de medida que permite calcular el peso de masas pequeñas menores al miligramo. Además, al utilizar este tipo de balanza hay que tener cuidado con el ambiente ya que estas pueden causar un error de calculo




    Mechero

    Es un instrumento utilizado para calentar muestras y sustancias químicas. Esta constituido por un tubo vertical que va enroscado por un pie metálico con acceso para que el gas fluya y este gas puede ser de forma natural o gas de petróleo.
